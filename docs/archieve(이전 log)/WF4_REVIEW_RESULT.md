# Task 1. 범용 Stage 0 프롬프트 초안

## 현재 프롬프트의 문제 진단

1. **Thinking Process 5축이 Bipolar+AUD에 하드코딩됨** — "기분장애 스펙트럼 내 감별", "물질 관련 장애와의 감별" 등은 다른 진단에서 무의미하거나 부적합
2. **Quality Gate의 "감별 진단 축 5개 모두 커버" 평가 기준** — 축 수가 진단에 따라 3~7개로 변동 가능
3. 나머지 요소(Role, Context, Constraints, Output Format)는 범용적이라 최소 수정만 필요

## 범용 Stage 0 프롬프트 전문

```
# Role
당신은 정신건강의학과 케이스 컨퍼런스 대비 전문 어시스턴트입니다.
임상경험 10년 이상의 정신건강의학과 전문의 수준의 지식을 보유하며,
레지던트가 교수 및 상급 레지던트로부터 받을 수 있는
**감별 진단 중심**의 질문을 사전 예측하고 방어 논리를 구성하는 데 특화됩니다.

# Context
- 사용자: 정신건강의학과 전공의 (R1~R4)
- 상황: 케이스 컨퍼런스 발표 직전 Q&A 대비
- 청중: 교수, 펠로우, 3-4년차 레지던트 (전문가 집단)
- 질문 패턴: **감별 진단 위주** — "왜 A이고, B/C/D는 왜 아닌가?"
- 핵심 원칙:
  - 없는 논문을 지어내는 것: **절대 금지**
  - 케이스에 없는 정보를 있는 것처럼 서술: **절대 금지**
  - 불확실한 문헌은 반드시 [확인 필요]로 표시
  - DSM-5-TR 기준은 직접 확인 가능한 기준만 인용. 기억이 불확실하면 "[DSM-5-TR 원문 확인 필요]"로 표기

# Task
입력된 Case Conference 보고서 전문을 분석하여, 감별 진단 중심의 예상 Q&A 대본을 생성하세요.

# Thinking Process — 동적 감별 축 도출 (5단계)

## Step 1: 진단 지형도 파악
보고서의 Diagnostic Formulation, Case Formulation, r/o 진단을 분석하여 아래 3 범주로 매핑합니다.

| 범주 | 정의 | 추출 방법 |
|------|------|----------|
| **확정 진단** | Diagnostic Formulation에 명시된 진단 | 보고서 VII 섹션 |
| **R/O 진단** | 보고서에 r/o로 명시되었거나, Case Formulation에서 감별 대상으로 암시된 진단 | VII + XI 섹션 |
| **누락 가능 진단** | 보고서에 없지만, Chief Problems + MSE + Personal History 조합상 교수가 공격할 수 있는 진단 | 증상-진단 교차 추론 |

## Step 2: 감별 축 자동 생성
Step 1의 결과를 바탕으로, 아래 규칙에 따라 감별 축을 **동적으로** 생성합니다.

**축 생성 규칙:**
1. 확정 진단 ↔ R/O 진단 각 쌍마다 감별 축 1개
2. 확정 진단 ↔ 누락 가능 진단 각 쌍마다 감별 축 1개
3. 확정 진단 간 증상 귀속이 모호한 경우 "증상 귀속 축" 추가
4. 삽화 특성(severity, specifier, 극성)에 대한 질문 가능성이 있으면 "삽화 특성 축" 추가
5. **유사 진단은 하나의 축으로 묶기**: 같은 진단 범주(예: Cluster B PD 3개)는 하나의 축으로 합산

**축 수 상한**: 최소 3개 ~ 최대 7개. 7개 초과 시 공격 예상도 하위 축을 통합.

## Step 3: 축별 예상 질문 생성
각 축에서 교수/상급 R이 던질 수 있는 질문을 생성합니다.

**질문 생성 우선순위:**
1. 보고서의 취약점(약한 근거, 모순, 누락)을 공격하는 질문
2. DSM-5-TR 기준 충족 여부를 따지는 질문
3. 시간축(onset, duration, 경과)의 정합성을 확인하는 질문
4. 보고서에 없는 감별 진단을 제기하는 질문

**위험도 부여 기준:**
- ⭐⭐⭐⭐⭐: 보고서에 결정적 취약점이 있어 방어 어려움, 또는 발표에서 누락된 핵심 감별
- ⭐⭐⭐⭐: 방어 가능하나 근거가 불충분하거나 불확실 영역 존재
- ⭐⭐⭐: 방어 논리가 명확하나, 알아두면 좋은 질문
- ⭐⭐: 나올 가능성 낮으나 대비 필요
- ⭐: 학술적으로 가능하나 실제 CC에서 잘 안 나오는 질문

## Step 4: DSM-5-TR 기준 대조 + 병력 근거 추출
각 질문에 대해:
- 해당 진단의 DSM-5-TR 기준을 표로 정리 (충족/미충족/불명확)
- 보고서의 PI, Personal History, MSE, Progress에서 직접 발췌한 근거 제시
- 지지 근거와 반론 근거를 균형 있게 나열

## Step 5: 불확실 영역 솔직 명시
- 현 시점에서 확정 불가능한 감별 과제를 식별
- 각 불확실 영역에 대해 추가 관찰/검사 계획 제시
- "인정 → 현재 근거 → 구체적 계획" 3단계 방어 프레임 적용

# Constraints — 생성하지 않는 질문 유형
- 치료 약물 선택 이유 (약물 A vs 약물 B)
- 약물 조합의 근거/위험성
- 검사 수치에 대한 처치 방향
- 감별 진단과 무관한 MSE 세부사항
- 예후에 대한 구체적 수치 예측

# Output Format

## 1. 진단 지형도 (표 형식)
확정/R/O/누락 가능 진단 전체 매핑. 각 진단별 공격 예상도(🔥🔥🔥~🔥🔥🔥🔥🔥).

## 2. 감별 축 목록 (표 형식)
자동 도출된 축 전체. 각 축의 핵심 쟁점 1줄 요약.

## 3. 핵심 취약점 (3~5가지)
보고서에서 방어가 가장 어려운 지점. 각 취약점에 대한 1줄 설명.

## 4. 예상 질문 목록 (위험도 순)
| 순위 | 질문 | 축 | 위험도 |
전체 15~20문항.

## 5. 상위 질문 전체 Q&A
각 질문에 대해 아래 형식으로 작성:

---
### ❓ Q[번호]. [질문]
**카테고리**: [축 이름] | **위험도** ⭐⭐⭐

**✅ 핵심 답변** (즉답 2-3문장)

**📋 DSM-5-TR 기준 대조** (표 형식)

**🔍 병력 근거** (보고서 직접 발췌, [PI], [Personal Hx], [MSE], [Progress] 출처 태깅)

**⚠️ 인정할 부분 / 추가 감별 계획**

**📖 추천 문헌** (확인 가능한 것만. 불확실하면 [확인 필요] 태깅)
---

## 6. 방어 문구 카드 (즉답용)
위험도 ⭐⭐⭐⭐ 이상 질문에 대해, R1이 3~5초 안에 말할 수 있는 핵심 1~2문장.

## 7. 불확실 영역 + 추가 계획
확정 불가 영역별로: 확정 불가 이유 → 감별 계획(시점별) → 대응 멘트

# Quality Gate
아래 4축 평균 8점 미만이면 재생성:
- 완전성(0-10): 도출된 감별 축 모두 커버되었는가?
- 정확성(0-10): DSM-5-TR 기준이 정확히 인용되었는가? (기억 불확실 시 "[확인 필요]" 표기 여부 포함)
- 실용성(0-10): R1이 즉답 가능한 형태인가?
- 근거(0-10): 병력에서 직접 발췌한 근거가 포함되었는가?
```

| **누락 가능** | Borderline Personality Disorder | identity disturbance, 유기불안, 자해위험, 대인관계 불안정 | 🔥🔥🔥🔥🔥 |
| **누락 가능** | Alcohol Use Disorder | 매일 맥주 2~6캔 × 4년 | 🔥🔥🔥🔥 |
| **누락 가능** | Substance-Induced Depressive Disorder | 음주와 우울 동시 경과 | 🔥🔥🔥 |
| **누락 가능** | Histrionic PD traits | 인정 욕구, 성애화, 과장된 감정 표현 | 🔥🔥 |

## Step 2: 감별 축 자동 도출 — 5개 축

| 축 | 핵심 쟁점 |
|----|----------|
| ① 우울장애 스펙트럼 내 감별 | MDD vs Persistent Depressive vs Adjustment Disorder |
| ② 양극성 스펙트럼 배제 | MDD vs Bipolar II (lithium 처방 이유, MDQ 시행 근거) |
| ③ 물질 관련 장애와의 감별 | MDD vs AUD vs Substance-Induced Depression (매일 2-6캔 × 4년) |
| ④ 성격장애와의 감별 | MDD vs BPD (identity disturbance가 Chief Problem #3) |
| ⑤ 증상 귀속 문제 | identity disturbance, 유기불안, 성애화 — MDD 증상인가 성격인가 |

**Contamination 검증**: "기분장애 스펙트럼 내 감별", "물질 관련 장애와의 감별" 같은 표현이 남아있으나, 이는 MDD 케이스에서도 유효한 축이다. Bipolar 특화 표현("삽화 특성 감별", "manic vs hypomanic")은 **자동으로 도출되지 않았다** — 범용화 성공.

## Step 3: 핵심 취약점

**취약점 1: AUD 누락** — Chief Problem에도, Diagnostic Formulation에도 AUD가 없음. "4년간 매일 맥주 2-6캔"은 DSM-5-TR AUD 기준 다수 충족 가능. 교수가 "왜 AUD를 진단에 넣지 않았나?"라고 공격할 가능성 매우 높음.

**취약점 2: Identity Disturbance를 Chief Problem에 넣은 근거** — identity disturbance는 MDD의 전형적 증상이 아니라 BPD의 핵심 기준. Chief Problem #3으로 올렸으면서 BPD를 감별 진단에서 언급하지 않은 것은 큰 공격 포인트.

**취약점 3: Bipolar 배제 근거 부족** — lithium을 처방하면서, MDQ 4개로 양극성을 어떻게 배제했는지 보고서에 설명 없음. "lithium 왜 쓰는데 MDD라고 하나?"라는 질문 불가피.

## Step 5: 상위 3개 질문 Q&A

### ❓ Q1. "Identity disturbance를 Chief Problem에 넣었는데, BPD는 왜 감별에 넣지 않았나요?"
**카테고리**: 성격장애와의 감별 | **위험도** ⭐⭐⭐⭐⭐

**✅ 핵심 답변**
BPD를 감별 진단에 포함하는 것이 적절했습니다. Identity disturbance는 DSM-5-TR BPD 9개 기준 중 #3에 해당하며, 이를 Chief Problem으로 올렸으면서 BPD를 감별에서 누락한 것은 보완이 필요합니다. 다만, 본 환자에서 BPD의 핵심인 반복적 자해/자살 행동 패턴(기준 #5)은 현재까지 확인되지 않으며, 대인관계 불안정의 양상도 MDD와 환경적 스트레서의 맥락에서 이해 가능합니다.

**📋 DSM-5-TR BPD 기준 대조**

| BPD 기준 | 본 환자 | 판단 |
|---|---|---|
| ① 유기를 피하기 위한 필사적 노력 | "버려지기 전에 버리자"는 심정으로 이혼 결심 → 유기불안 존재 | ❓ 가능 |
| ② 불안정한 대인관계(이상화-평가절하) | 남편 평가절하 + 직장 상사 이상화 | ✅ |
| ③ 정체성 장애 | Chief Problem #3으로 명시 | ✅ |
| ④ 충동성 (≥2영역) | 음주, 외도 관계 | ✅ |
| ⑤ 반복적 자해/자살 행동 | SA(-), 자해력 없음. SI는 있으나 반복적 행동화 없음 | ✖ |
| ⑥ 정서 불안정 | 우울 위주, 급격한 반응적 변동은 두드러지지 않음 | ❓ |
| ⑦ 만성적 공허감 | "기생충 같다" — 공허감보다는 자존감 저하에 가까움 | ❓ |
| ⑧ 부적절한 분노 | 남편에게 "쏘아붙인" 에피소드 있으나, 빈도·강도 제한적 | ❓ |
| ⑨ 일과성 편집증/해리 | 외도 사건에 대한 la belle indifference (해리적) | ❓ |
| **충족** | | **확실 3개 + 불명확 4개** |

**🔍 병력 근거**
- [Chief Problem] Identity disturbance를 #3 주소로 명시
- [PI] "버려지기 전에 버리자는 심정으로 처음 이혼을 결심" → 유기불안 패턴
- [PI] 직장 상사와의 관계: "예쁨 받는 느낌이 그립다", "위안을 얻고" → 이상화·의존 패턴
- [Psychodynamic] "외디푸스적 갈망을 포기하지 않고 부적절한 상대를 선택하여 삼각관계 양상을 재현"
- [MSE] SI(+)이나 SA(-), 자해력 없음 → BPD 핵심 기준 #5 미충족

**⚠️ 인정할 부분**
BPD를 감별에 포함하지 않은 것은 보완이 필요합니다. 다만 BPD full diagnosis보다는 trait 수준으로 판단하며, 이유는: (1) 반복적 자해/자살 행동이 없고, (2) 대인관계 불안정이 환경 스트레서(남편 갈등) 맥락에 한정되어 있으며, (3) MDD가 BPD 증상을 악화시키는 방향으로도 설명 가능합니다. 퇴원 후 MDD 치료 반응 관찰 및 대인관계 패턴 추적으로 최종 판단할 계획입니다.

### ❓ Q2. "4년간 매일 맥주 2-6캔이면 AUD 아닌가요? 왜 진단에 없나요?"
**카테고리**: 물질 관련 장애와의 감별 | **위험도** ⭐⭐⭐⭐⭐

**✅ 핵심 답변**
지적에 동의합니다. 4년간 매일 맥주 2-6캔 음주는 DSM-5-TR AUD 기준 다수를 충족할 가능성이 높으며, 진단에 포함하는 것이 적절했습니다. Lipase 상승(61)도 알코올 관련 소견입니다. AUD를 동반 진단으로 추가하는 것을 고려하겠습니다.

**📋 DSM-5-TR AUD 기준 점검**

| AUD 기준 | 본 환자 | 판단 |
|---|---|---|
| 1. 의도한 것보다 많이/오래 사용 | 매일 2-6캔 (범위 큼 → 조절 어려움 시사) | ✅ 가능 |

| 4. 갈망 | 명시적 기술 없음 | ❓ |
| 5. 역할 수행 실패 | "1달째 집안일 못함" | ✅ |
| 6. 사회적 문제에도 지속 | 남편 갈등 중에도 지속 | ✅ |
| 10. 내성 | 2캔→6캔 증가 | ✅ |
| **예상 충족** | | **4~5개 = Moderate AUD** |

**🔍 병력 근거**
- [Past Hx] "4년간 매일 맥주 2~6캔 음주"
- [Lab] Lipase ↑61 — 알코올 관련 가능
- [PI] "뱃살이 많이 쪘다는 생각에 놀라 원래 매일 마시던 술을 절반 이하로 줄였다" → 사용량 인식은 있으나 완전 중단 아님

**⚠️ 인정할 부분**
AUD를 진단에서 누락한 것은 보고서의 약점입니다. Comorbid AUD로 추가하고, substance-induced depressive disorder와의 감별도 논의에 포함할 필요가 있습니다.

### ❓ Q3. "Lithium을 쓰면서 MDD라고 진단한 근거는? BD II 가능성은?"
**카테고리**: 양극성 스펙트럼 배제 | **위험도** ⭐⭐⭐⭐⭐

**✅ 핵심 답변**
Lithium은 MDD에서도 항우울제 augmentation으로 사용됩니다. 본 환자는 escitalopram 단독 치료에 반응이 불충분하여 lithium augmentation이 추가된 것으로 판단됩니다. MDQ 4개로 양극성 선별은 음성(cutoff 7개)이며, MSE에서 manic/hypomanic 증상은 관찰되지 않습니다. 다만 MDQ의 민감도 한계(약 70-80%)를 고려하면 BD II 위음성 가능성을 완전히 배제할 수 없으며, 치료 경과 중 hypomanic episode 발현 여부를 모니터링할 필요가 있습니다.

**🔍 병력 근거**
- [Past Hx] escitalopram 10mg + lithium 450mg → augmentation 패턴
- [Scale] MDQ 4개 < cutoff 7개 → 양극성 선별 음성
- [MSE] mood: depressed, speech: moderate, grandiosity(-), flight of idea(-) → 현재 manic/hypomanic 소견 없음
- [PI] 별거 기간 "다른 남성에게 예쁨 받는 느낌이 그립다" → impulsive 행동이나, 에너지 증가·수면 감소 등 manic feature 동반 기술 없음

**⚠️ 인정할 부분**
Lithium augmentation이 BD 가능성을 암시하는 것은 아니지만, 처방 근거를 보고서에 명시하지 않은 것은 약점입니다. BD II의 depressive episode는 MDD와 감별이 어려우며, 생애 첫 hypomanic episode가 아직 발현하지 않았을 가능성도 있습니다. 외래 추적 중 mood charting으로 모니터링 예정입니다.

## 상위 5개 질문 Q&A

### ❓ Q1. "기분 증상이 폭음과 동시인데, Substance-Induced BD 아닌가요?"
**카테고리**: 물질 관련 장애와의 감별 | **위험도** ⭐⭐⭐⭐⭐

**✅ 핵심 답변**
Substance-Induced BD 배제를 위해서는 (1) 물질 사용 이전에 기분삽화가 선행했거나 (2) 물질 중단 후 약 1개월 이상 증상 지속이 필요합니다. (1)은 대학시절 삽화가 가능하나 당시에도 음주 동반으로 불명확합니다. (2)는 금주 5일째 금단은 호전 추세이나 기분 증상(hyperthymic, pressured speech, grandiosity)은 지속되어 두 증상의 경과가 분리되고 있어 독립적 삽화를 시사합니다. 최종 확인을 위해 4주 단주 관찰이 필수적입니다.

**📋 DSM-5-TR 감별 기준 대조**

| 기준 | 본 환자 | 판단 |
|---|---|---|
| 물질 사용 이전 기분삽화 선행 | 대학시절 가능하나 당시도 음주 동반 | 불명확 |
| 물질 중단 후 1개월+ 증상 지속 | 금주 5일째 조증 지속, 1개월 미달 | 관찰 중 |
| 물질의 알려진 효과로 설명 불가 | 지속적 grandiosity, pressured speech는 intoxication과 다름 | 시사적 |
| 독립적 기분장애 병력 | 대학시절 hypomanic episode 가능 | 부분 충족 |

**🔍 병력 근거**

독립적 BD 지지:
- [Personal Hx] "성인이 되었다는 생각에 자유롭고 행복해" → 기분 고양이 선행, 음주는 뒤따르는 행동
- [Progress HD#2~#5] 금주 상태에서 hyperthymic mood, pressured speech, grandiosity 지속
- [Progress] 금단(seizure, tremor)은 호전 → 기분 증상은 변화 없음 → 경과 분리

Substance-Induced 지지:
- [PI] 기분 onset과 폭음 악화 모두 내원 3개월 전 동시
- [Personal Hx] 대학시절에도 heavy drinking 동반
- [Family Hx] BD 가족력 없음, AUD 가족력만 존재(친조부, 친백부)

**⚠️ 인정할 부분**: 가장 핵심적 감별 과제임을 인정. 4주 단주 시점에서 YMRS 재평가 예정. 과거 단주 시기(임신·수유기 등) 기분 상태를 보호자와 추가 면담 예정.

### ❓ Q2. "술 없이 기분삽화가 있었던 적이 한번이라도 있나요?"
**카테고리**: 물질 관련 장애와의 감별 | **위험도** ⭐⭐⭐⭐⭐

**✅ 핵심 답변**
현재까지 확인된 범위에서는 완전 단주 상태의 독립적 기분삽화는 명확히 확인되지 않습니다. 대학시절 삽화도 음주와 동반되어 있었습니다. 이것이 가장 큰 약점이며, 임신·수유기 등 단주 시기의 기분 상태를 보호자와 추가 면담하여 확인할 계획입니다.

**🔍 병력 근거**
- [Personal Hx] 대학시절: "자주 술을 마시고 클럽에... 인사불성이 될 때까지" → 기분 고양과 음주 공존
- [PI] 기분 증상 악화 시기(내원 3개월 전)에도 "아침저녁으로 맥주... 소주 3병... 폭음"
- 임신·수유기: 보고서에 해당 시기 기분 상태 기술 없음 → 추가 면담 필요

**⚠️ 인정할 부분**: 이 질문에 대한 완전한 방어는 현시점에서 불가능합니다. "선생님 지적에 동의합니다. 순수 단주 삽화 확인이 아직 안 되어 있으며, 보호자 추가 면담과 4주 단주 관찰로 확인할 계획입니다."

### ❓ Q3. "BPD 아닌가요?"
**카테고리**: 성격장애와의 감별 | **위험도** ⭐⭐⭐⭐⭐

**✅ 핵심 답변**
BPD를 감별에 포함해야 한다는 점에 동의합니다. 다만 BD I을 우선 진단으로 둔 이유는: (1) 현재 행동 변화가 보호자 관찰상 "이런 적은 처음"이라는 episodic 양상, (2) BPD 핵심인 만성적·반복적 자해/자살 행동이 없음, (3) 기분 불안정 양상이 BPD(수시간 내 반응적 변동)보다 BD(수일~수주 지속 삽화적 고양)에 더 부합합니다.

**📋 DSM-5-TR BPD 기준 대조**

| BPD 기준 | 본 환자 | 판단 |
|---|---|---|
| ① 유기 회피 필사적 노력 | 분노/보복 반응이 우세 | ✖ |
| ② 불안정한 대인관계 | 남편 평가절하 + 내연남 이상화 → 삽화적 | ❓ |
| ③ 정체성 장애 | 기술 없음 | ✖ |
| ④ 충동성 (≥2영역) | 음주, 성적 행동 → 삽화적 악화 | ✅ |
| ⑤ 반복적 자해/자살 행동 | SI(-), SA(-), 자해력 없음 | ✖ |
| ⑥ 정서 불안정 | Labile mood(+) | ✅ |
| ⑦ 만성적 공허감 | 기술 없음 | ✖ |
| ⑧ 부적절한 분노 | 욕설/막말 → 보호자 "이런 적은 처음" | ❓ |
| ⑨ 일과성 편집증/해리 | Paranoid ideation(+) | ❓ |
| **충족** | | **약 3~4개 (5개 미달)** |

**🔍 병력 핵심 근거 — Episodic vs Pervasive**
- [Progress HD#2 보호자] "이런 적은 처음이에요" / "이정도는 아니었어요. 지금은 제정신이 아닌거같아요"
- [Identifying Data] 친구 20명, 속마음 친구 3명 → 안정적 우정 유지
- 생애 전반 자해·자살 시도 없음

**⚠️ 인정할 부분**: 아동기부터의 충동성, 정서적 반응성은 BPD trait 수준으로 존재 가능성 인정. BD와 BPD 공존도 가능하며, 삽화 관해 후 성격 패턴 지속 여부 관찰 필요.

### ❓ Q4. "충동성은 어디에 귀속시키나요?"
**카테고리**: 증상 귀속 문제 | **위험도** ⭐⭐⭐⭐⭐

**✅ 핵심 답변**
시기와 질적 수준으로 분리합니다. (1) 아동기 충동성(떼쓰기, 절도, 자기주장 관철) → 성격 trait. (2) 내원 3개월 전~현재(클럽 성관계, 한 달 가출, 자녀 방치, 이혼 선언) → 핵심 생활 영역 파괴 수준으로 삽화적(episodic). (3) 알코올 → 악화 인자이나, 금주 상태에서도 impaired judgment + aggressive behavior 지속으로 알코올만으로는 불충분.

**🔍 병력 근거**
- [Progress HD#2 보호자] "원래도 고집불통에 통제 안되는 성격이긴 했는데 이정도는 아니었어요"
- 입원 후(금주 상태): aggressive behavior(+), impaired judgment(+) → 알코올 제거 후 지속

### ❓ Q5. "BD I이 아니라 MDD with mixed features 아닌가요?"
**카테고리**: 기분장애 스펙트럼 내 감별 | **위험도** ⭐⭐⭐⭐⭐

**✅ 핵심 답변**
MDD with mixed features는 주요우울삽화 기간 중 조증 증상이 동반되는 경우이며, 독립적 manic episode 기준 충족 시 BD I이 우선 진단됩니다. 본 환자는 내원 1개월 전 이후 우울 없이(BDI 6점) hyperthymic mood, pressured speech, grandiosity가 지속되어 독립적 manic episode에 부합합니다.

**🔍 병력 근거**
- [Progress] 입원 후 HD#2~#5: mood hyperthymic 일관, BDI 6(normal), pressured speech 지속
- [PI] 내원 1개월 전: 내연남과 "6시간 대화가 끊이지 않고", "단칸방에서 시작해도 괜찮다" → MDE 소견 없이 manic feature만
- DSM-5-TR 위계: manic episode 충족 시 BD I 우선

# Task 4. 임상 타당성 검토

## 1. 감별 축의 임상적 타당성

**GS1 (MDD)**: 자동 도출된 5축 중 "BPD 감별"과 "AUD 감별"은 실제 CC에서 가장 먼저 공격받을 포인트와 정확히 일치한다. Identity disturbance를 Chief Problem으로 올리면서 BPD를 감별에 안 넣은 것은 교수가 반드시 지적할 지점이며, 이를 잡아낸 것은 유효하다.

**누락된 공격 포인트**: GS1에서 "외도 관계의 임상적 의미" — 단순 MDD 증상으로 보기엔 성애화(sexualization) 패턴이 두드러진다. Psychodynamic formulation에서는 다루었으나, 감별 진단 질문에서 "이 행동 패턴이 MDD만으로 설명되는가?"라는 질문이 빠졌다. 이는 축 ⑤(증상 귀속)에서 다뤄야 하나 현재 시뮬레이션에서는 깊이가 부족했다.

**GS2 (Bipolar+AUD)**: 5축 모두 실전 CC와 일치. 특히 "Substance-Induced vs Independent"가 최대 격전지라는 판단은 대본 4.1과 동일하며 임상적으로 정확하다.

## 2. 질문 현실성

**CC에서 나오지 않을 질문**: 자동 생성본에서는 특별히 비현실적인 질문이 없다. 다만 "Cyclothymic Disorder"나 "Alcohol-Related Neurocognitive Disorder" 같은 대본 4.1의 마이너 감별은 자동 생성에서 누락되었는데, 이는 실제 CC에서도 잘 안 나오는 질문이므로 문제되지 않는다.

**주의점**: 치료 질문 제외 constraint가 잘 작동하고 있다. "왜 aripiprazole이고 olanzapine이 아닌가?" 같은 치료 질문은 생성되지 않았음.

## 3. 즉답 가능성

자동 생성본의 "핵심 답변"은 2-3문장으로, R1이 3-5초 안에 핵심 논지를 말할 수 있는 수준이다. 다만 **대본 4.1의 "대응 멘트(🎤)" 형태가 더 실용적**이다. 자동 생성본에서는 핵심 답변이 약간 학술적 톤이고, 대본 4.1의 멘트는 실제 CC에서 입으로 말하는 형태("선생님 지적에 동의합니다. 다만...")로 더 자연스럽다.

**프롬프트 보완 권고**: Output Format에 "방어 멘트 카드" 섹션을 명시적으로 추가하고, "실제 CC에서 입으로 말하는 톤으로 작성"이라는 지시를 추가해야 한다.

## 4. Hallucination 위험 영역

가장 높은 위험:
1. **DSM-5-TR 기준 세부 수치/기간** — 예: "hypomanic episode는 4일 이상"을 "3일 이상"으로 잘못 인용할 위험. 현재 프롬프트의 "[DSM-5-TR 원문 확인 필요]" 지시가 이를 부분적으로 방어하지만, 모델이 자신 있게 잘못 인용하는 경우를 완전히 막지는 못함.

2. **논문 인용** — 현재 Stage 0에서는 논문을 직접 인용하지 않도록 제한되어 있어 위험 낮음. Perplexity 보강 단계에서 위험 급증.

3. **"보호자가 ~라고 했다"의 과잉 해석** — 보호자 발언을 정확히 인용하는 것은 잘하지만, 발언의 임상적 의미를 과잉 해석하는 경우가 있을 수 있다. 예: "에너지가 넘쳐보여요"를 manic episode 근거로 쓰되, 이것이 보호자의 주관적 표현임을 명시하지 않는 문제.

# Task 6. 종합 판정 및 권고

## 1. 범용 프롬프트 판정: **Conditional Pass**

GS1(MDD)과 GS2(Bipolar+AUD) 양쪽에서 유효한 감별 축을 동적 도출했으며, Bipolar 특화 잔재 없이 범용화에 성공했다.

**필수 수정 사항 (Pass 전제조건)**:
1. **Output Format에 "방어 멘트 카드" 섹션 추가** — 위험도 ⭐⭐⭐⭐+ 질문에 대해 "🎤 대응 멘트: 선생님 지적에 동의합니다..." 형태의 즉답 카드 생성 지시
2. **"증상-진단 교차 매핑 표" 생성 지시 추가** — 대본 4.1의 가장 유용한 부분. 보고서의 Chief Problems × 감별 진단을 교차 표로 정리
3. **불확실 영역 + 추가 계획을 "타임라인 표"로 강제** — "즉시/2주/4주/관해 후/외래" 시점별 계획

## 2. WF4 구현 우선순위: 기획안 W4-02(Tier 5 완료 후) **적절함**

Tier 5(GS2 contamination-free 검증)가 WF4의 입력물(완성 보고서)의 품질을 결정한다. 보고서 품질이 75점+ 미달이면 Q&A 대본의 품질도 보장할 수 없다. 순서 변경 불필요.

## 3. Option B(Perplexity 논문 보강) 타당성: **Option A 우선, B는 조건부**

| 기준 | Option A (Stage 0만) | Option B (Stage 0 + Perplexity) |
|------|---------------------|-------------------------------|
| 비용 | ~$0.3~0.5 | ~$1.0~1.5 |
| 품질 | 대본 4.1의 70~90% | 논문이 정확하면 +10~15%, 부정확하면 신뢰도 하락 |
| Hallucination 위험 | 낮음 (DSM-5만 참조) | 중~높음 (논문 DOI 검증 불가 시) |

## 4. 기획안 v0.1의 최대 리스크 Top 3

**리스크 1: DSM-5-TR 기준 Hallucination**
LLM이 DSM-5-TR 기준을 정확히 인용하는지에 대한 검증 체계가 없다. "hypomanic episode 4일 이상"을 "3일"로 잘못 쓰면 R1이 그대로 외워서 CC에서 오답. 대응: (1) 프롬프트에 "[기억 불확실 시 확인 필요 태깅]" 이미 포함했으나, (2) 생성 후 자동 검증 노드(DSM-5 기준 체크리스트 대조) 추가를 고려해야 한다. 현 기획안에 이 검증 단계가 없다.

**리스크 2: 범용 프롬프트의 진단 커버리지 한계**
GS1(MDD)과 GS2(Bipolar+AUD) 두 케이스만으로 검증했다. Schizophrenia, PTSD, OCD, Eating Disorder 등 다른 주요 진단에서 동적 축 도출이 실패할 가능성이 있다. 특히 psychotic disorders에서는 "양성증상 vs 음성증상 vs 인지 증상" 같은 다른 차원의 감별이 필요한데, 현 프롬프트가 이를 커버하는지 미검증.

**리스크 3: Perplexity 논문 Hallucination의 cascade 효과**
Perplexity가 존재하지 않는 논문을 생성하면, AI Agent가 이를 Q&A에 삽입하고, R1이 CC에서 인용할 위험. "[확인 필요]" 태깅만으로는 R1이 확인을 실제로 안 할 가능성이 높다. 대응: Stage 0에서 논문 인용을 완전히 제외하고, 논문이 필요하면 사용자가 수동으로 Perplexity/Gemini Deep Research를 돌리는 것이 더 안전.
